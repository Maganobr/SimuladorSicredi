Durante o processo desenvolvi um conhecimento abrangente acerca de testes automatizados. 

Analisando a história, do ponto de vista do negócio, vi que temos muitos pontos que podemos validar, desde o perfil do usuário até o tempo de investimento, escolhi focar nos valores a serem investidos, tendo em vista o perfil de pessoa física somente e em uma contagem de período em meses apenas.  

Decidi testar em Java utilizando Junit e Selenium, utilizando o Maven e Eclipse que são ferramentas que já havia tido contato previamente, durante a faculdade. 

Ao longo do projeto tive algumas dificuldades pois como havia utilizado as ferramentas há anos atrás, não estava habituado nem a utilizar uma IDE de desenvolvimento. 

Mapiei os cenários da seguinte maneira: 

Investimentos válidos: 

Investimento mínimo;

Investimento médio;

Investimento máximo.

Investimentos inválidos: 

Valor aplicado inválido;

Valor investido inválido; 

Tempo de investimento inválido.

Como nunca trabalhei profissionalmente com testes automatizados não sabia exatamente se era necessário um output dos resultados, portanto não o implementei. 

Muito obrigado pela oportunidade.
